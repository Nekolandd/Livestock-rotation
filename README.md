# Segundo Examen Parcial - II Parte

Proyecto desarrollado para la segunda parte del Segundo Examen Parcial del curso **Diseño y Programación de Plataformas Móviles (EIF411)**.  
I Ciclo - 2026.

## Descripción
Prototipo funcional de una aplicación móvil en Android para el control de rotación de ganado en lotes de pasturas de fincas (potreros). Utiliza persistencia local con SQLite a través de Room ORM.
